## Objective

Add a new Accounts tab to the tab bar: an Expo Router route mounting a new `accounts-screen`, registered between Home and Journal, with a localized title and an icon.

## Context

- Tab registration: `app/(app)/(tabs)/_layout.tsx` (Ionicons icons; titles come from i18n keys refreshed on `localeVar` change — follow the existing `tabTitles` pattern).
- Route files pattern: `app/(app)/(tabs)/journal.tsx` mounting `src/screens/journal-screen/`.
- Screen conventions: `SafeAreaView`, `usePageView("accounts")` analytics, `LedgerGuard`/`useLedgerGuard` like `src/screens/ledger-screen/ledger-screen.tsx`.
- New string `accounts` goes in the English base `src/translations/en.ts`; other locales fall back.

## Steps

1. Create `src/screens/accounts-screen/accounts-screen.tsx` skeleton (SafeAreaView, themed background, LedgerGuard, page-view analytics) + `index.ts` export.
2. Create route `app/(app)/(tabs)/accounts.tsx` mounting it.
3. Register the tab in `app/(app)/(tabs)/_layout.tsx` after Home with an Ionicons icon (e.g. `cash`) and `accounts` title in the `tabTitles` state.
4. Add the `accounts` key to `src/translations/en.ts`.
