## Objective

Replace the two separate stacked charts with a single **swipeable account-charts card** (top of the dashboard, like Monarch's net-worth card) that pages across: (1) **Net worth** line chart, (2) **Spending by category** sum, (3) **Liabilities** sum. Page 1 mirrors the current net-worth line; pages 2–3 are the "swipe to category sum / liabilities sum" the request asks for.

## Context

- Carousel from t002 (`src/components/paged-carousel/`), card chrome from t001 (`src/components/dashboard-card/`).
- Existing charts: `src/common/d3/line-chart-d3.tsx` (`LineChartD3`), `src/common/d3/bar-chart-d3.tsx` (`BarChartD3`) — both take `{ labels, numbers, currencySymbol }`.
- Data + selectors already in the home screen:
  - `useHomeCharts` → `lastSixWorthData` (net worth series) — page 1.
  - `useAccountHierarchy` → `accounts` (assets/liabilities/income/expenses/equity totals) via `src/screens/home-screen/selectors/select-account-totals.ts`.
  - The raw `AccountHierarchy` expenses node has `children[]` with per-account `balance_children` — the source for a **spending-by-category** breakdown (page 2).
- GraphQL: `src/common/graphql/queries/accountHierarchy.graphql`, `homeCharts.graphql` (already generated — do **not** hand-edit `src/generated-graphql/`).

## Steps

1. Add `src/screens/home-screen/selectors/select-spending-by-category.ts`: from the `AccountHierarchy` "expenses" node, produce top-N categories with their summed balances (`{ labels, numbers }`) in the active currency, folding the remainder into an "Other" bucket. Handle string/number and missing-currency cases like `select-account-totals.ts` does.
2. Build `src/screens/home-screen/components/account-charts-card.tsx`: a `DashboardCard` wrapping a `PagedCarousel` with three pages — net worth (`LineChartD3` on `lastSixWorthData`), spending-by-category (`BarChartD3` on the new selector), liabilities (a summary/`BarChartD3` or big-number view of the liabilities total from `select-account-totals`). Show the headline value + label for the active page.
3. Mount it at the top of `home-screen.tsx`, replacing the two standalone chart sections. Keep loading skeletons (`LoadingTile`) with background colors while `netWorthLoading`/`accountsLoading`.
4. Add category/liabilities/net-worth page labels to `src/translations/en.ts`.
