## Objective

Create the single source of truth for typography: a `typography` module in the theme exposing the mono font family, a font-size scale, weight constants (regular/medium only), and a shared amount style with `fontVariant: ["tabular-nums"]`.

## Context

- `src/common/theme/index.ts` holds only color themes today; no typography tokens exist anywhere.
- Current drift to converge: 151 `fontSize` declarations across `src/screens` + `src/components`; 6 duplicated `fontVariant: ["tabular-nums"]` styles in `src/screens/add-transaction-screen/*`; `MONO_FONT` defined locally in `src/screens/journal-screen/journal-bottom-sheet/sheet-content.tsx`.
- Derive the size scale from the actual distribution (`grep -rn "fontSize" src/`) — cluster the 151 values into ~6 steps rather than inventing numbers; mono minimum is 13.
- Until t002 lands, `fonts.mono` resolves via `Platform.select({ ios: "Menlo", default: "monospace" })` so the token is correct standalone.

## Steps

1. Add `src/common/theme/typography.ts`: `fonts` (mono family), `fontSizes` scale, `fontWeights` (regular, medium), `amountStyle` (tabular-nums + mono-min-size guard where used for mono amounts).
2. Export it from `src/common/theme/index.ts`.
3. Short doc comment stating the rules: system font for prose, mono for structured ledger text ≥13, tabular figures for amounts.
