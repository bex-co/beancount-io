# Ledger change history via listCommits/getCommitDetails

Read-only "what changed" screen: `listCommits` + `getCommitDetails` (both in the mobile schema, unused). Shows recent commits to the ledger — useful trust/fidelity feature now that mobile writes (quick add, m8 multi-leg, m9 edits, m10 receipts) mutate the ledger. Entry point: ledger drawer. Sub-hour for a list; drill-down to diffs would need scoping.
